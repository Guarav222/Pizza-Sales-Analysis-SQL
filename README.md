
# 🍕 Pizza Sales Analysis using MySQL

## 📌 Project Overview

This project analyzes pizza sales data using **MySQL** by solving real-world business problems. It demonstrates SQL skills such as joins, aggregate functions, subqueries, and window functions to extract meaningful business insights from sales data.

---

## 🎯 Objective

The objective of this project is to analyze pizza sales data and answer business-related questions to understand:

- Customer purchasing behavior
- Sales performance
- Revenue trends
- Popular pizza categories and sizes
- Peak ordering hours

---

## 🗂 Dataset

The project uses four related datasets:

| Table Name | Description |
|------------|-------------|
| Orders | Stores order date and order time |
| Order_Details | Stores quantity of pizzas ordered |
| Pizzas | Stores pizza size and price |
| Pizza_Types | Stores pizza names and categories |

## 📊 Business Questions Solved

1. Retrieve the total number of orders placed.
2. Calculate total revenue generated from pizza sales.
3. Identify the highest-priced pizza.
4. Find the most common pizza size ordered.
5. List the top 5 most ordered pizza types.
6. Find total quantity ordered by pizza category.
7. Determine order distribution by hour.
8. Find category-wise pizza distribution.
9. Calculate the average number of pizzas ordered per day.
10. Determine the top 3 pizza types based on revenue.
11. Calculate the percentage contribution of each pizza type to total revenue.
12. Determine the top 3 revenue-generating pizzas within each category.

## 🚀 How to Run

1. Open MySQL Workbench.
2. Create a new database.
3. Import the CSV files into MySQL tables.
4. Run the SQL queries from `Pizza_Sales_SQL_Queries.sql`.
5. Analyze the results and compare them with the project presentation.
